# 🎬 Cinemaverse

A modern Netflix-style movie browsing app built with **Next.js 15**, **Tailwind CSS**, and **TMDB API**.

## 🚀 Features
- Browse popular movies
- Responsive UI with Tailwind
- Dynamic routing for movie pages
- Navbar and MovieCard components

## 📦 Stack
- Next.js 15 (App Router)
- Tailwind CSS
- TMDB API
- Axios, SWR

## 🧪 Running Locally

```bash
git clone https://github.com/solufelo/Cinemaverse.git
cd Cinemaverse
npm install
npm run dev
